Vocaloid

Vocaloid (ボーカロイド) is software developed by Yamaha. It is a Virtual Singing Synthesizer that allows for realistic sounding singing voices using virtual voicebanks.

About the Software

Vocaloid 6 is the latest version of Yamaha's "singing synthesizer", It introduces an AI engine, multilingual singing—allowing seamless mixing of Japanese, English, Chinese with native-like pronunciation in the same track. Along with advanced and improved editing tools for dynamics, vibrato, and vocal layering, it also remains compatible with older Vocaloid "Voicebanks".
